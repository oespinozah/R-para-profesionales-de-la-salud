# Datos de ejemplo

Esta carpeta contiene datos simulados o archivos mínimos usados por los capítulos del manual.

No son bases reales de pacientes ni materiales de evaluación. Su propósito es que una persona pueda ejecutar los ejemplos de importación, descriptivos e inferencia sin depender de archivos externos.

## Archivos

- `analisis-descriptivo.xlsx`: base simulada para estadística descriptiva.
- `ejemplo-descriptivo.xlsx`: archivo simple generado en el capítulo de descriptivos.
- `inferencia-datos-simulados.xlsx`: base simulada para ejemplos de inferencia.
- `importacion/`: versiones pequeñas en distintos formatos para demostrar importación y exportación con `rio`.
